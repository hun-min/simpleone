# 대기 중인 작업 (2025-01-XX)

## 1. 자동완성 복원
- **문제**: 자동완성이 사라짐
- **요구사항**: 
  - 맨 처음 카드 만들 때도 자동완성 뜨게
  - 팝업 편집 때도 자동완성 뜨게
- **상태**: ❌ 미완료

## 2. 전체보기 클릭 동작 수정
- **문제**: 전체보기에서 클릭하면 팝업이 안 뜨고 다른 게 뜸
- **요구사항**: 전체보기에서도 카드 클릭 시 TaskDetailPopup 띄우기
- **상태**: ❌ 미완료

## 3. 프로토콜 단계 2번 텍스트 고정
- **문제**: "찬물 마시기"가 누를 때마다 바뀜
- **요구사항**: "🔥 각성 하기"로 고정
- **상태**: ❌ 미완료

## 4. 팝업 ESC 닫기
- **문제**: 팝업에서 ESC 눌러도 안 꺼짐
- **요구사항**: 모든 팝업에서 ESC 누르면 닫히게
- **상태**: ❌ 미완료

## 5. 팝업 드래그 방지
- **문제**: 팝업 켠 상태에서 텍스트 드래그해서 바깥으로 나가서 떼면 꺼짐
- **요구사항**: 드래그 시작이 팝업 내부면 바깥에서 떼도 안 꺼지게
- **상태**: ✅ 완료 - popupMouseDownTarget 사용해서 드래그 시작 위치 체크

## 6. 텍스트 수정창 높이
- **문제**: 텍스트 수정창이 아직도 큼
- **요구사항**: 카드 높이만큼 (24px)
- **상태**: ❌ 미완료

## 7. 모든 사항 기록
- **요구사항**: 모든 사항을 기록해놓고 사용자가 물어보면 대답할 수 있게
- **상태**: ✅ 완료 - 이 파일로 구현됨

## 8. 하위할일 숫자 표시 위치
- **문제**: 하위할일 완료 숫자를 카드에 표기해버림
- **요구사항**: 상세 팝업에서만 표기해야 함
- **상태**: ❌ 미완료 - 카드에 표기하지 말고 상세 팝업에만 표기

## 9. 완료 버튼 위치
- **문제**: 완료 버튼이 팝업 왼쪽에 있음
- **요구사항**: 시작 버튼 왼쪽에 두기
- **상태**: ❌ 미완료 - 시작 버튼 바로 왼쪽에 배치해야 함

## 현재 상태 요약
- ✅ 완료: 5번(팝업 드래그 방지), 7번(기록 시스템)
- ❌ 미완료: 1번(자동완성), 2번(전체보기 클릭), 3번(프로토콜 텍스트), 4번(ESC 닫기), 6번(텍스트 높이), 8번(하위할일 표시 위치), 9번(완료 버튼 위치)
