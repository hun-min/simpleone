## 2. 전체보기 클릭 동작 수정
- **문제**: 전체보기에서 클릭하면 팝업이 안 뜨고 다른 게 뜸
- **요구사항**: 전체보기에서도 카드 클릭 시 TaskDetailPopup 띄우기
- **상태**: ✅ 완료 - toggleTimer → setTaskDetailPopup으로 변경

## 3. 프로토콜 단계 2번 텍스트 고정
- **문제**: "찬물 마시기"가 누를 때마다 바뀜
- **요구사항**: "🔥 각성 하기"로 고정
- **상태**: ✅ 완료 - "찬물 한 컵을 마시세요!"로 고정

## 4. 팝업 ESC 닫기
- **문제**: 팝업에서 ESC 눌러도 안 꺼짐
- **요구사항**: 모든 팝업에서 ESC 누르면 닫히게
- **상태**: ✅ 완료 - 모든 팝업에 ESC 키 이벤트 추가

## 5. 팝업 드래그 방지

- **상태**: ✅ 완료 - popupMouseDownTarget 사용해서 드래그 시작 위치 체크

## 6. 텍스트 수정창 높이

- **상태**: ✅ 완료 - minHeight 제거, height: 24px로 고정
